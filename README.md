#  Sustainable_AI

Sustainable  AI  –  Transparency  and  Energy-Efficient  Prompt/Context  Engineering  with  Machine  Learning

##  Setup  Instructions

Follow  the  steps  below  to  set  up  and  run  the  **Sustainable  AI  Prompt  Optimizer**  project.

---

###  1.  Prerequisites

-  Python  **3.11.3**  installed    
-  `git`  installed  (optional  but  recommended)    
-  A  virtual  environment  tool  (`venv`  or  Conda)

You  can  verify  your  Python  version  with:

  
python  --version
#  or
python3  --version
Here’s  a  cleaned-up  and  structured  version  you  can  drop  into  your  `README.md`:

###  2.  Create  and  Activate  a  Virtual  Environment

####  Windows  (PowerShell  /  CMD)

  
python  -m  venv  .venv
.\.venv\Scripts\activate
  

####  macOS  /  Linux

  
python3  -m  venv  .venv
source  .venv/bin/activate
  

You  should  now  see  `(.venv)`  at  the  start  of  your  terminal  prompt.

---

###  3.  Install  Dependencies

Make  sure  you  are  in  the  project  root  (where  `requirements.txt`  is  located),  then  run:

  
pip  install  --upgrade  pip
pip  install  -r  requirements.txt
